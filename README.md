# 📈 Stock Market Data Analyzer

## 📌 Project Overview

The **Stock Market Data Analyzer** is a Python-based financial analytics project that collects historical stock market data using Yahoo Finance and performs trend analysis, moving average analysis, volatility analysis, and financial data visualization.

This project helps investors, analysts, and students understand stock price behavior through data-driven insights and charts.

---

# 🚀 Features

✅ Historical stock market data collection using Yahoo Finance  
✅ Data cleaning and preprocessing  
✅ Daily return calculation  
✅ Moving average analysis (MA20 & MA50)  
✅ Volatility analysis  
✅ Highest and lowest stock price analysis  
✅ Financial data visualization  
✅ Correlation heatmap  
✅ Candlestick chart visualization  
✅ Final summary report generation  
✅ Google Colab implementation
